# Modulo Palavro

Modulo isolado do jogo Palavro dentro do portal.

## Estrutura

- `index.html`: entrada da pagina do jogo
- `src/`: interface, motor e cliente da API
- `data/`: palavras aceitas e respostas
- `backend/netlify/functions/`: funcoes serverless do jogo

## Observacao

O desafio diario e centralizado no backend e usa o horario oficial de Brasilia.
